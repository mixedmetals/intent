# Accordion

Collapsible sections for organizing content.

## Example

<Accordion 
  items={[
    {
      id: '1',
      title: 'What is Intent Framework?',
      content: 'Intent is a schema-first, AI-native CSS framework that bridges the gap between design intent and implementation. It uses schemas to define components and generates optimized CSS at build time.'
    },
    {
      id: '2',
      title: 'How do I get started?',
      content: 'Install the intent-core and intent-react packages, import the theme CSS, and start using components. Check out our Quick Start guide for more details.'
    },
    {
      id: '3',
      title: 'Is Intent production ready?',
      content: 'Yes! Intent is used in production by several teams. The framework provides 72 production-ready components with full TypeScript support and accessibility features.'
    }
  ]}
  style={{ marginBottom: 24 }}
/>

```tsx
import { Accordion } from 'intent-react'

<Accordion 
  items={[
    {
      id: '1',
      title: 'What is Intent Framework?',
      content: 'Intent is a schema-first CSS framework...'
    },
    {

      title: 'How do I get started?',
```

  allowMultiple
  items={[

    },

```tsx
<Accordion allowMultiple items={[...]} />
```

## Default Open Items

```tsx
<Accordion 
  defaultOpen={['1', '2']}
  items={[...]}
/>
```

## Controlled Component

```tsx
import { useState } from 'react'
import { Accordion } from 'intent-react'

function ControlledAccordion() {
  const [openItems, setOpenItems] = useState(['1'])
  
  return (
    <Accordion 
      openItems={openItems}
      onChange={setOpenItems}
      items={[...]}
    />
  )
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `Array<{id: string, title: string, content: ReactNode}>` | `[]` | Accordion items |
| `allowMultiple` | `boolean` | `false` | Allow multiple open items |
| `defaultOpen` | `string[]` | `[]` | Initially open item IDs |
| `openItems` | `string[]` | - | Controlled open items |
| `onChange` | `(items: string[]) => void` | - | Change callback |
| `size` | `'sm'` \| `'md'` \| `'lg'` | `'md'` | Accordion size |

## When to Use

Use **Accordion** for:
- FAQs
- Settings panels with many sections
- Hiding detailed content
- Mobile-friendly content organization

Use **Tabs** when:
- Content is at the same level of importance
- Users need to switch frequently
- Space is limited

# AlertDialog

Modal dialogs for critical alerts requiring user action. Uses the **Elevated** archetype.

:::tip
AlertDialog is a specialized Modal variant for destructive or important actions.
:::

export const AlertDialogDemo = () => {

## Props

AlertDialog uses the same props as Modal with additional recommendations:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | - | Alert title |
| `description` | `string` | - | Alert description |
| `onConfirm` | `() => void` | - | Confirm callback |
| `confirmText` | `string` | `'Continue'` | Confirm button text |
| `cancelText` | `string` | `'Cancel'` | Cancel button text |
| `destructive` | `boolean` | `false` | Use error color for confirm |

## Design Notes

- Uses **Elevated** archetype (floating above content)
- Destructive actions use error color
- Clear cancel/confirm separation
- No close button (must choose an action)
- Focus trap for accessibility

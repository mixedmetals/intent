# Alert

Important messages that attract user attention. Uses the **Flat** archetype with status-based colors.

## Examples

### Variants

<div className="intent-demo">
  <Stack gap="normal">
    <Alert status="info" title="Did you know?">
      Alerts can contain helpful information about your application.
    </Alert>
    <Alert status="success" title="Success!">
      Your changes have been saved successfully.
    </Alert>
    <Alert status="warning" title="Heads up">
      Your subscription expires in 3 days.
    </Alert>
    <Alert status="error" title="Error">
      Unable to save changes. Please try again.
    </Alert>
  </Stack>
</div>

```tsx
import { Alert, Stack } from 'intent-react';

<Stack gap="normal">
  <Alert status="info" title="Did you know?">
    Alerts can contain helpful information.
  </Alert>
  <Alert status="success" title="Success!">
    Your changes have been saved.
  </Alert>
  <Alert status="warning" title="Heads up">
    Your subscription expires soon.
  </Alert>
  <Alert status="error" title="Error">
    Unable to save changes.
  </Alert>
</Stack>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `status` | `'info' \| 'success' \| 'warning' \| 'error'` | `'info'` | Status/color variant |
| `variant` | `'subtle' \| 'solid' \| 'left-accent' \| 'top-accent'` | `'subtle'` | Visual style |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Alert size |
| `title` | `string` | - | Optional title |
| `dismissible` | `boolean` | `false` | Show dismiss button |
| `onDismiss` | `() => void` | - | Dismiss handler |
| `children` | `ReactNode` | - | Alert content |

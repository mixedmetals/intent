# Alert

Important messages that attract user attention.

export const Alert = ({ children, variant = 'info', title }) => (
  <div className={`intent-alert intent-alert--${variant}`}>
    <div className="intent-alert__body">
      {title && <div className="intent-alert__title">{title}</div>}
      <div style={{ fontSize: '13px', lineHeight: 1.5 }}>{children}</div>
    </div>
  </div>
)

export const Stack = ({ children, gap = 'normal' }) => (
  <div className={`intent-stack intent-stack--direction-column intent-stack--gap-${gap}`}>
    {children}
  </div>
)

## Examples

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'info'` \| `'success'` \| `'warning'` \| `'danger'` | `'info'` | Alert style |
| `title` | `string` | - | Alert heading |

## When to Use

| Variant | Use Case |
|---------|----------|
| **Info** | General information, tips |
| **Success** | Completed actions, confirmations |
| **Warning** | Attention needed, non-blocking issues |
| **Danger** | Errors, blocking issues |

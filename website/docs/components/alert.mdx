# Alert

Important messages that attract user attention.

## Examples

### Variants

<Stack gap="normal" style={{ marginBottom: 24 }}>

  </Alert>

  </Alert>

    Your subscription expires in 3 days.
  </Alert>
  <Alert variant="danger" title="Error">
    Unable to save changes. Please try again.
  </Alert>
</Stack>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'info'` \| `'success'` \| `'warning'` \| `'danger'` | `'info'` | Alert style |
| `title` | `string` | - | Alert heading |
| `dismissible` | `boolean` | `false` | Show close button |
| `compact` | `boolean` | `false` | Reduced padding |
| `onDismiss` | `() => void` | - | Dismiss callback |
| `icon` | `string` \| `false` | - | Custom icon or hide |

## When to Use

| Variant | Use Case |
|---------|----------|
| **Info** | General information, tips |
| **Success** | Completed actions, confirmations |
| **Warning** | Attention needed, non-blocking issues |
| **Danger** | Errors, blocking issues |

# AspectRatio

Maintain consistent aspect ratios for media. Uses the **Flat** archetype.

## Examples

### Video (16:9)

<div className="intent-demo" style={{ maxWidth: '400px' }}>
  <AspectRatio ratio={16 / 9}>
    <div style={{ 
      width: '100%', 
      height: '100%', 
      background: 'linear-gradient(135deg, #667eea 0%, #764ba2 100%)',
      display: 'flex',
      alignItems: 'center',
      justifyContent: 'center',
      color: 'white',
      fontWeight: 600
    }}>
      16:9 Video Placeholder
    </div>
  </AspectRatio>
</div>

```tsx
import { AspectRatio } from 'intent-react/components/layout';

<AspectRatio ratio={16 / 9}>
  <video src="..." />
</AspectRatio>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `ratio` | `number` | `16/9` | Aspect ratio (width/height) |
| `children` | `ReactNode` | - | Content to constrain |

## Common Ratios

| Ratio | Use Case |
|-------|----------|
| `16/9` | Video, widescreen |
| `4/3` | Classic TV, photos |
| `1` | Square images, avatars |
| `3/4` | Portrait photos |
| `21/9` | Ultrawide video |

## Design Notes

- Uses padding-bottom technique for fluid ratio
- Child content is absolutely positioned
- Common for images, videos, and embeds
- Prevents layout shift during load

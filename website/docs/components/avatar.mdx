# Avatar

User profile image with fallback initials. Uses the **Flat** archetype.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | - | Image URL |
| `alt` | `string` | - | Alt text for image |
| `initials` | `string` | - | Fallback text (max 2 chars) |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl'` | `'md'` | Avatar size |
| `variant` | `'circle' \| 'square' \| 'rounded'` | `'circle'` | Shape |
| `color` | `'neutral' \| 'primary' \| 'success' \| 'warning' \| 'error'` | `'neutral'` | Initials background |

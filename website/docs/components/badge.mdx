import { Badge, Stack } from 'intent-react';

# Badge

Small status indicators for UI elements. Uses the **Flat** archetype with subtle tints.

## Examples

### Variants

<div className="intent-demo">
  <Stack direction="row" gap="normal">
    <Badge>Default</Badge>
    <Badge variant="primary">Primary</Badge>
    <Badge variant="success">Success</Badge>
    <Badge variant="warning">Warning</Badge>
    <Badge variant="danger">Danger</Badge>
  </Stack>
</div>

```tsx
import { Badge } from 'intent-react';

<Badge>Default</Badge>
<Badge variant="primary">Primary</Badge>
<Badge variant="success">Success</Badge>
<Badge variant="warning">Warning</Badge>
<Badge variant="danger">Danger</Badge>
```

### Usage Examples

<div className="intent-demo">
  <Stack direction="row" gap="normal" align="center">
    <span>New features <Badge variant="primary">Beta</Badge></span>
    <span>Status: <Badge variant="success">Active</Badge></span>
    <span>Priority: <Badge variant="danger">High</Badge></span>
  </Stack>
</div>

```tsx
<span>New features <Badge variant="primary">Beta</Badge></span>
<span>Status: <Badge variant="success">Active</Badge></span>
<span>Priority: <Badge variant="danger">High</Badge></span>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'primary' \| 'success' \| 'warning' \| 'danger'` | `'default'` | Visual style |
| `children` | `ReactNode` | - | Badge content |

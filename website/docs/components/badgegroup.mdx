# BadgeGroup

Group of badges with overflow handling. Uses the **Flat** archetype.

:::tip
For simple badge groups, use `HStack` with multiple `Badge` components.
:::

## Examples

### Basic Group

<div className="intent-demo">
  <HStack gap="compact" wrap>
    <Badge>Design</Badge>
    <Badge variant="success">Ready</Badge>
    <Badge variant="warning">Review</Badge>
    <Badge variant="error">Blocked</Badge>
  </HStack>
</div>

```tsx
import { Badge, HStack } from 'intent-react';

<HStack gap="compact" wrap>
  <Badge>Design</Badge>
  <Badge variant="success">Ready</Badge>
  <Badge variant="warning">Review</Badge>
  <Badge variant="error">Blocked</Badge>
</HStack>
```

### With Limit

<div className="intent-demo">
  <HStack gap="compact" wrap>
    <Badge>React</Badge>
    <Badge>TypeScript</Badge>
    <Badge>CSS</Badge>
    <Badge>+3 more</Badge>
  </HStack>
</div>

```tsx
const tags = ['React', 'TypeScript', 'CSS', 'Node.js', 'GraphQL', 'PostgreSQL'];
const visible = tags.slice(0, 3);

<HStack gap="compact" wrap>
  {visible.map(tag => <Badge key={tag}>{tag}</Badge>)}
  {tags.length > 3 && <Badge>+{tags.length - 3} more</Badge>}
</HStack>
```

### Removable Badges

<div className="intent-demo">
  <HStack gap="compact" wrap>
    <Badge>JavaScript <button style={{ marginLeft: '4px', cursor: 'pointer' }}>×</button></Badge>
    <Badge>Python <button style={{ marginLeft: '4px', cursor: 'pointer' }}>×</button></Badge>
    <Badge>Rust <button style={{ marginLeft: '4px', cursor: 'pointer' }}>×</button></Badge>
  </HStack>
</div>

```tsx
const [badges, setBadges] = useState(['JavaScript', 'Python', 'Rust']);

const removeBadge = (badge) => {
  setBadges(badges.filter(b => b !== badge));
};

<HStack gap="compact" wrap>
  {badges.map(badge => (
    <Badge key={badge}>
      {badge}
      <button onClick={() => removeBadge(badge)}>×</button>
    </Badge>
  ))}
</HStack>
```

## Props

### BadgeGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `badges` | `Array<{id: string, label: string, variant?: string}>` | `[]` | Badge items |
| `limit` | `number` | - | Maximum badges to show |
| `removable` | `boolean` | `false` | Allow removing badges |
| `onRemove` | `(id: string) => void` | - | Remove callback |

## Design Notes

- Uses **Flat** archetype for badges
- Consistent spacing between badges
- Overflow with "+N more" indicator
- Close button for removable badges
- Wraps to multiple lines if needed

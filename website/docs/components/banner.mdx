# Banner

Persistent notification at the top of the page.

## Example

```tsx
import { useState } from 'react'
import { Banner } from 'intent-react'

function Example() {
  const [show, setShow] = useState(true)
  
  if (!show) return null
  
  return (
    <Banner 
      variant="info"
      title="New Features Available"
      description="Check out the latest updates."
      action={{ label: 'Learn More', onClick: () => {} }}
      onDismiss={() => setShow(false)}
    />
  )
}
```

## Variants

### Info

```tsx
<Banner 
  variant="info"
  title="Scheduled Maintenance"
  description="The system will be down for maintenance on Sunday at 2 AM."
/>
```

### Success

```tsx
<Banner 
  variant="success"
  title="Deployment Successful"
  description="Your changes are now live."
/>
```

### Warning

```tsx
<Banner 
  variant="warning"
  title="Storage Almost Full"
  description="You've used 90% of your storage. Upgrade to add more."
  action={{ label: 'Upgrade', onClick: handleUpgrade }}
/>
```

### Danger

```tsx
<Banner 
  variant="danger"
  title="Payment Failed"
  description="We couldn't process your payment. Please update your billing info."
  action={{ label: 'Update Billing', onClick: handleBilling }}
/>
```

## Sticky

Keep banner visible when scrolling:

```tsx
<Banner 
  sticky
  variant="info"
  title="Welcome!"
  description="Complete your profile to get started."
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'info'` \| `'success'` \| `'warning'` \| `'danger'` | `'info'` | Banner style |
| `title` | `string` | required | Banner title |
| `description` | `string` | - | Banner description |
| `action` | `{ label: string, onClick: () => void }` | - | Action button |
| `dismissible` | `boolean` | `true` | Show dismiss button |
| `onDismiss` | `() => void` | - | Dismiss callback |
| `sticky` | `boolean` | `false` | Stick to top |

## Banner vs Alert vs Toast

Use **Banner** when:
- Information is persistent and important
- At the top of the page
- User should eventually read it

Use **Alert** when:
- Contextual to a specific section
- Immediate feedback needed

Use **Toast** when:
- Temporary notification
- Doesn't require immediate action

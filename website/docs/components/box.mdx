# Box

Single element wrapper for spacing, borders, and backgrounds. Generic layout primitive. Uses the **Flat** archetype.

## Examples

### Basic Box

<div className="intent-demo">
  <div style={{ padding: '16px', border: '1px solid var(--intent-color-border)', borderRadius: '8px', background: 'var(--intent-color-surface)' }}>
    Box provides a simple container with padding.
  </div>
</div>

```tsx
import { Box } from 'intent-react/components/layout';

<Box padding="normal" border>
  Content goes here
</Box>
```

### Display Variants

<div className="intent-demo">
  <div style={{ display: 'flex', flexDirection: 'column', gap: '8px' }}>
    <div style={{ padding: '16px', background: 'var(--intent-color-surface-elevated)', borderRadius: '8px' }}>
      Block display (default)
    </div>
    <div style={{ padding: '16px', background: 'var(--intent-color-surface-elevated)', borderRadius: '8px', display: 'flex' }}>
      Flex display
    </div>
    <div style={{ padding: '16px', background: 'var(--intent-color-surface-elevated)', borderRadius: '8px', display: 'inline-block' }}>
      Inline block
    </div>
  </div>
</div>

```tsx
<Box display="block">Block display</Box>
<Box display="flex">Flex display</Box>
<Box display="inline-block">Inline block</Box>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `display` | `'block' \| 'inline' \| 'inline-block' \| 'flex' \| 'inline-flex' \| 'grid' \| 'none'` | `'block'` | Display type |
| `position` | `'static' \| 'relative' \| 'absolute' \| 'fixed' \| 'sticky'` | `'static'` | Position type |
| `width` | `'auto' \| 'full' \| 'screen' \| '1/2' \| '1/3' \| '1/4' \| ...` | `'auto'` | Width |
| `height` | `'auto' \| 'full' \| 'screen' \| '1/2' \| '1/3' \| ...` | `'auto'` | Height |
| `padding` | `'none' \| 'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'none'` | Padding |
| `overflow` | `'visible' \| 'hidden' \| 'scroll' \| 'auto'` | `'visible'` | Overflow behavior |
| `zIndex` | `'auto' \| '0' \| '10' \| '20' \| '30' \| '40' \| '50'` | `'auto'` | Z-index |

## Design Notes

- Uses **Flat** archetype (no bevel)
- Generic layout primitive
- All CSS properties available as props
- Supports responsive values
- No opinionated styling

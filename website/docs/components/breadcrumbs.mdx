# Breadcrumbs

Navigation breadcrumbs with enhanced styling. Shows the current page location within a hierarchy.

## Examples

### Basic Breadcrumbs

<div className="intent-demo">
  <Breadcrumbs>
    <BreadcrumbItem href="/">Home</BreadcrumbItem>
    <BreadcrumbItem href="/docs">Docs</BreadcrumbItem>
    <BreadcrumbItem current>Components</BreadcrumbItem>
  </Breadcrumbs>
</div>

```tsx
import { Breadcrumbs, BreadcrumbItem } from 'intent-react/components/navigation';

<Breadcrumbs>
  <BreadcrumbItem href="/">Home</BreadcrumbItem>
  <BreadcrumbItem href="/docs">Docs</BreadcrumbItem>
  <BreadcrumbItem current>Components</BreadcrumbItem>
</Breadcrumbs>
```

## Props

### Breadcrumbs

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `separator` | `'chevron' \| 'slash' \| 'arrow' \| 'dot'` | `'chevron'` | Separator style |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Breadcrumb size |

### BreadcrumbItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `href` | `string` | - | Link URL (omit for current page) |
| `current` | `boolean` | `false` | Mark as current/active page |

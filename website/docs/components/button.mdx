# Button

Buttons trigger actions or navigate to different pages.

## Overview

The Button component uses the **Milled** archetype with radial gradient backgrounds, light-catcher borders, and inset shadows for a tactile, pressed-metal aesthetic.

export const Button = ({ children, importance = 'secondary', size = 'medium', fullWidth, disabled, ...props }) => (
  <button 
    className={`intent-button intent-button--importance-${importance} intent-button--size-${size} ${fullWidth ? 'intent-button--full-width' : ''}`}
    disabled={disabled}
    {...props}
  >
    {children}
  </button>
)

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `importance` | `'primary' \| 'secondary' \| 'tertiary'` | `'secondary'` | Visual weight of the button |
| `size` | `'small' \| 'medium' \| 'large'` | `'medium'` | Size of the button |
| `fullWidth` | `boolean` | `false` | Whether to expand to full width |
| `disabled` | `boolean` | `false` | Whether the button is disabled |

## Design Tokens

Buttons use the **Milled** surface archetype:

| Token | Value | Purpose |
|-------|-------|---------|
| Background | `radial-gradient(120% 80% at 50% 0%, oklch(100% 0 0 / 0.12), transparent 70%), oklch(25% 0.01 260)` | Radial gradient surface |
| Border | `oklch(100% 0 0 / 0.12)` top, `oklch(0% 0 0 / 0.3)` bottom | Light-catcher effect |
| Shadow | `inset 0 1px 0 rgba(255,255,255,0.08), 0 2px 4px rgba(0,0,0,0.15)` | Inset highlight + drop shadow |
| Border Radius | `8px` (`--intent-radius-md`) | Rounded corners |

# Calendar

Date picker calendar component. Uses the **Flat** archetype for day selection states.

export const CalendarDemo = () => {
  const [selected, setSelected] = React.useState(new Date())
  return (
    <Calendar 
      selectedDate={selected} 
      onSelect={(date) => setSelected(date)}
    />
  )
}

### Date Range

export const RangeDemo = () => {
  const today = new Date()
  const nextWeek = new Date(today.getFullYear(), today.getMonth(), today.getDate() + 7)
  
  return (
    <Calendar 
      selectedDate={[today, nextWeek]}
      minDate={today}
      maxDate={new Date(today.getFullYear(), today.getMonth() + 2, 0)}
    />
  )
}

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size of calendar |
| `view` | `'month' \| 'year' \| 'decade'` | `'month'` | Current view mode |
| `currentDate` | `Date` | - | Controlled displayed month/year |
| `selectedDate` | `Date \| Date[]` | - | Selected date(s) |
| `minDate` | `Date` | - | Minimum selectable date |
| `maxDate` | `Date` | - | Maximum selectable date |
| `onSelect` | `(date: Date) => void` | - | Callback when date is selected |
| `onMonthChange` | `(date: Date) => void` | - | Callback when month changes |
| `renderDay` | `(date, props) => ReactNode` | - | Custom day renderer |

## Design Notes

- Days use the **Flat** archetype (no bevel)
- Selected state uses `var(--intent-color-primary)` background
- Today is indicated with a ring accent
- Hover states use `var(--intent-color-surface-hover)`
- Navigation buttons use the **Milled** archetype for tactile interaction

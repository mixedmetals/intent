# Card

Content container with elevation and padding options.

## Examples

### Basic Card

<div style={{ marginBottom: 24 }}>
  <Card padding="normal">
    <Text>Card content goes here. Use cards to group related information.</Text>
  </Card>
</div>

```tsx
<Card padding="normal">
  <Text>Card content goes here.</Text>
</Card>
```

### Interactive Card

<div style={{ marginBottom: 24 }}>
  <Card 
    elevation="low" 
    padding="normal" 
    interactive
    onClick={() => alert('Card clicked!')}
  >
    <Text weight="bold">Clickable Card</Text>
    <Text size="sm" color="muted">This card responds to clicks and hover states.</Text>
  </Card>
</div>

```tsx
<Card 
  elevation="low" 
  padding="normal" 
  interactive
  onClick={handleClick}
>
  <Text weight="bold">Clickable Card</Text>
  <Text size="sm" color="muted">This card responds to clicks.</Text>
</Card>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `elevation` | `'none'` \| `'low'` \| `'medium'` \| `'high'` | `'medium'` | Shadow depth |
| `padding` | `'none'` \| `'compact'` \| `'normal'` \| `'relaxed'` | `'normal'` | Inner padding |
| `interactive` | `boolean` | `false` | Enable hover/focus states |
| `onClick` | `() => void` | - | Click handler |

## CSS Output

```css
.card {
  background: var(--intent-color-surface);
  border-radius: var(--intent-radius-lg);
  border: 1px solid var(--intent-color-border);
}

.card--elevation-low {
  box-shadow: var(--intent-shadow-sm);
}

.card--elevation-medium {
  box-shadow: var(--intent-shadow-md);
}

.card--interactive:hover {
  box-shadow: var(--intent-shadow-lg);
  transform: translateY(-1px);
}
```

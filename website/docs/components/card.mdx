# Card

Content container with elevation and padding options. Uses the **Elevated** archetype with backdrop blur, layered shadows, and hairline borders.

## Examples

### Basic Card

<div className="intent-demo">
  <Card padding="normal">
    <Text>Card content goes here. Use cards to group related information.</Text>
  </Card>
</div>

```tsx
import { Card, Text } from 'intent-react';

<Card padding="normal">
  <Text>Card content goes here.</Text>
</Card>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `elevation` | `'none' \| 'low' \| 'medium' \| 'high'` | `'medium'` | Shadow depth |
| `padding` | `'none' \| 'compact' \| 'normal' \| 'relaxed'` | `'normal'` | Inner padding |
| `interactive` | `boolean` | `false` | Enable hover/focus states |
| `onClick` | `() => void` | - | Click handler |

# Center

Center content horizontally and vertically using flexbox.

## Examples

### Basic Center

<div className="intent-demo">
  <Center style={{ minHeight: '120px' }}>
    <div style={{ padding: '16px 32px', background: 'var(--intent-color-primary)', color: 'white', borderRadius: '8px', fontWeight: 600 }}>
      I'm centered!
    </div>
  </Center>
</div>

```tsx
import { Center } from 'intent-react/components/layout';

<Center>
  <div>Centered content</div>
</Center>
```

### Column Direction

<div className="intent-demo">
  <Center direction="column" style={{ minHeight: '150px', gap: '12px' }}>
    <div style={{ fontSize: '24px', fontWeight: 'bold' }}>Title</div>
    <div style={{ color: 'var(--intent-color-text-secondary)' }}>Subtitle</div>
    <button style={{ padding: '8px 16px', marginTop: '8px' }}>Action</button>
  </Center>
</div>

```tsx
<Center direction="column">
  <h1>Title</h1>
  <p>Subtitle</p>
  <button>Action</button>
</Center>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `inline` | `boolean` | `false` | Use inline-flex instead of flex |
| `direction` | `'row' \| 'row-reverse' \| 'column' \| 'column-reverse'` | `'row'` | Flex direction |
| `children` | `ReactNode` | - | Content to center |

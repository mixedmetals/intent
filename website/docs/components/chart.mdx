# Chart

Data visualization component.

## Examples

### Line Chart

```tsx
import { Chart } from 'intent-react'

<Chart
  type="line"
  data={{
    labels: ['Jan', 'Feb', 'Mar', 'Apr', 'May'],
    datasets: [{
      label: 'Revenue',
      data: [12000, 19000, 15000, 25000, 22000],
    }]
  }}
/>
```

### Bar Chart

```tsx
<Chart
  type="bar"
  data={{
    labels: ['Product A', 'Product B', 'Product C'],
    datasets: [{
      label: 'Sales',
      data: [300, 450, 200],
    }]
  }}
  options={{
    colors: ['#3b82f6'],
  }}
/>
```

### Pie Chart

```tsx
<Chart
  type="pie"
  data={{
    labels: ['Desktop', 'Mobile', 'Tablet'],
    datasets: [{
      data: [60, 30, 10],
    }]
  }}
  options={{
    colors: ['#3b82f6', '#10b981', '#f59e0b'],
  }}
/>
```

### Area Chart

```tsx
<Chart
  type="area"
  data={{
    labels: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri'],
    datasets: [{
      label: 'Visitors',
      data: [120, 190, 150, 250, 220],
    }]
  }}
  fill
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `'line'` \| `'bar'` \| `'pie'` \| `'area'` \| `'doughnut'` | required | Chart type |
| `data` | `ChartData` | required | Chart data |
| `options` | `ChartOptions` | `{}` | Chart options |
| `height` | `number` | `300` | Chart height |
| `width` | `number` | - | Chart width |
| `fill` | `boolean` | `false` | Fill area (area charts) |

### ChartData Type

```typescript
interface ChartData {
  labels: string[]
  datasets: {
    label?: string
    data: number[]
    color?: string
  }[]
}
```

### ChartOptions Type

```typescript
interface ChartOptions {
  colors?: string[]
  legend?: boolean
  grid?: boolean
  tooltips?: boolean
  animations?: boolean
}
```

# Container

Centered, max-width wrapper for page content.

## Examples

### Sizes

<div className="intent-demo">
  <Container size="small" style={{ marginBottom: '16px' }}>
    <Card style={{ textAlign: 'center' }}>Small (640px)</Card>
  </Container>
  <Container size="medium" style={{ marginBottom: '16px' }}>
    <Card style={{ textAlign: 'center' }}>Medium (768px)</Card>
  </Container>
  <Container size="large" style={{ marginBottom: '16px' }}>
    <Card style={{ textAlign: 'center' }}>Large (1024px)</Card>
  </Container>
</div>

```tsx
import { Container } from 'intent-react/components/layout';

<Container size="small">...</Container>
<Container size="medium">...</Container>
<Container size="large">...</Container>
<Container size="xl">...</Container>
```

### Padding

<div className="intent-demo">
  <Container size="medium" padding="compact">
    <Card>Compact padding (4px)</Card>
  </Container>
</div>

```tsx
<Container padding="compact">...</Container>
<Container padding="normal">...</Container>
<Container padding="relaxed">...</Container>
```

### Fluid Container

```tsx
<Container fluid>Full width with padding</Container>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'small' \| 'medium' \| 'large' \| 'xl' \| 'fluid'` | `'medium'` | Max width size |
| `padding` | `'compact' \| 'normal' \| 'relaxed'` | `'normal'` | Horizontal padding |
| `center` | `boolean` | `true` | Center the container |

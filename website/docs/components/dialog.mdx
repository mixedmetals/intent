# Dialog

Accessible modal dialogs. Uses the **Elevated** archetype.

:::tip
Dialog is an alias for Modal. Use whichever name fits your mental model.
:::

<div className="intent-demo">

```tsx
import { Modal } from 'intent-react';

const [isOpen, setIsOpen] = useState(false)

<Button onClick={() => setIsOpen(true)}>Open Dialog</Button>

<Modal 
  isOpen={isOpen} 
  onClose={() => setIsOpen(false)}
  title="Edit Profile"
  description="Make changes to your profile"
>
  <Form>
    <Input label="Name" />
    <Textarea label="Bio" />
  </Form>
  <ModalFooter>
    <Button importance="secondary">Cancel</Button>
    <Button>Save</Button>
  </ModalFooter>
</Modal>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `isOpen` | `boolean` | `false` | Whether dialog is open |
| `onClose` | `() => void` | - | Called when dialog closes |
| `title` | `string` | - | Dialog title |
| `description` | `string` | - | Dialog description |
| `children` | `ReactNode` | - | Dialog content |
| `footer` | `ReactNode` | - | Footer content |

## Design Notes

- Uses **Elevated** archetype (floating above content)
- Larger shadow than Card to indicate higher elevation
- Backdrop uses `rgba(0,0,0,0.5)` with blur
- Header and footer have subtle 1px borders
- Focus trap and escape key handling

# Divider

Visual separator between content sections.

## Examples

### Horizontal Divider

<div className="intent-demo">
  <div style={{ display: 'flex', flexDirection: 'column', gap: '16px' }}>
    <div>Content above</div>
    <Divider />
    <div>Content below</div>
  </div>
</div>

```tsx
import { Divider } from 'intent-react';

<div>Content above</div>
<Divider />
<div>Content below</div>
```

### With Label

<div className="intent-demo">
  <div style={{ display: 'flex', flexDirection: 'column', gap: '16px' }}>
    <button className="intent-button intent-button--importance-primary intent-button--size-medium" style={{ width: '100%' }}>Sign in with Email</button>
    <Divider label="OR" />
    <button className="intent-button intent-button--importance-secondary intent-button--size-medium" style={{ width: '100%' }}>Sign in with Google</button>
  </div>
</div>

```tsx
<Divider label="OR" />
```

### Vertical

<div className="intent-demo">
  <div style={{ display: 'flex', alignItems: 'center', gap: '16px', height: '60px' }}>
    <span>Left</span>
    <Divider orientation="vertical" />
    <span>Right</span>
  </div>
</div>

```tsx
<div style={{ display: 'flex', alignItems: 'center', gap: '16px' }}>
  <Text>Left</Text>
  <Divider orientation="vertical" />
  <Text>Right</Text>
</div>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Divider direction |
| `label` | `string` | - | Center label text |

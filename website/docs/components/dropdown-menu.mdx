# DropdownMenu

Dropdown menus with submenus and keyboard navigation. The menu uses the **Elevated** archetype while menu items use the **Flat** archetype.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `trigger` | `ReactNode` | - | Trigger element |
| `items` | `Array<{label?: string, icon?: ReactNode, shortcut?: string, onClick?: () => void, disabled?: boolean, separator?: boolean}>` | `[]` | Menu items |

## Design Notes

- Menu container uses **Elevated** archetype (floating above content)
- Menu items use **Flat** archetype (hover state only)
- Separators use 1px `var(--intent-color-border)`
- Disabled items have 0.5 opacity
- Shortcuts aligned to the right with muted color

# Dropdown

Menu that appears when clicking a trigger element.

## Example

```tsx
import { Button, Dropdown } from 'intent-react'

<Dropdown 
  trigger={<Button>Options</Button>}
  items={[
    { label: 'Edit', value: 'edit', icon: 'edit' },
    { label: 'Duplicate', value: 'duplicate', icon: 'copy' },
    { label: 'Delete', value: 'delete', icon: 'trash', danger: true },
  ]}
  onSelect={(item) => console.log(item)}
/>
```

## With Sections

```tsx
<Dropdown 
  trigger={<Button>View Options</Button>}
  items={[
    { 
      section: 'Layout',
      items: [
        { label: 'Grid view', value: 'grid', icon: 'grid' },
        { label: 'List view', value: 'list', icon: 'list' },
      ]
    },
    { 
      section: 'Sort by',
      items: [
        { label: 'Name', value: 'name' },
        { label: 'Date', value: 'date' },
        { label: 'Size', value: 'size' },
      ]
    },
  ]}
/>
```

## With Disabled Items

```tsx
<Dropdown 
  trigger={<Button>Actions</Button>}
  items={[
    { label: 'Copy', value: 'copy', icon: 'copy' },
    { label: 'Paste', value: 'paste', disabled: true },
    { separator: true },
    { label: 'Delete', value: 'delete', danger: true },
  ]}
/>
```

## Alignment

```tsx
<Dropdown align="start" trigger={...} items={...} />
<Dropdown align="center" trigger={...} items={...} />
<Dropdown align="end" trigger={...} items={...} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `trigger` | `ReactNode` | required | Element that opens the dropdown |
| `items` | `Array<Item>` | `[]` | Dropdown items |
| `align` | `'start'` \| `'center'` \| `'end'` | `'start'` | Menu alignment |
| `side` | `'top'` \| `'right'` \| `'bottom'` \| `'left'` | `'bottom'` | Menu position |
| `open` | `boolean` | - | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | - | Open state callback |
| `onSelect` | `(item: Item) => void` | - | Item select callback |

### Item Type

```typescript
interface Item {
  label: string
  value: string
  icon?: string
  danger?: boolean
  disabled?: boolean
  separator?: boolean
  section?: string
  items?: Item[]  // For nested sections
}
```

## Dropdown vs Select

Use **Dropdown** for:
- Action menus
- Context menus
- Command palettes
- Overflow menus

Use **Select** for:
- Form inputs
- Picking from a list of values
- When the selection needs to be displayed

# Field

Label and validation wrapper for form inputs.

## Examples

### Basic Field

<div className="intent-demo">
  <div style={{ display: 'flex', flexDirection: 'column', gap: '4px' }}>
    <Label htmlFor="email">Email Address</Label>
    <Input id="email" type="email" placeholder="you@example.com" />
  </div>
</div>

```tsx
import { Field, Input } from 'intent-react';

<Field label="Email Address">
  <Input type="email" placeholder="you@example.com" />
</Field>
```

### With Hint

<div className="intent-demo">
  <div style={{ display: 'flex', flexDirection: 'column', gap: '4px' }}>
    <Label htmlFor="username">Username</Label>
    <Input id="username" placeholder="johndoe" />
    <span style={{ fontSize: '0.75rem', color: 'var(--intent-color-muted)' }}>Must be unique and 3-20 characters</span>
  </div>
</div>

```tsx
<Field label="Username" hint="Must be unique and 3-20 characters">
  <Input placeholder="johndoe" />
</Field>
```

### With Error

<div className="intent-demo">
  <div style={{ display: 'flex', flexDirection: 'column', gap: '4px' }}>
    <Label htmlFor="password">Password</Label>
    <Input id="password" type="password" validation="invalid" />
    <span style={{ fontSize: '0.75rem', color: 'var(--intent-color-error)' }}>Password must be at least 8 characters</span>
  </div>
</div>

```tsx
<Field 
  label="Password" 
  error="Password must be at least 8 characters"
>
  <Input type="password" validation="invalid" />
</Field>
```

### Required Field

<div className="intent-demo">


## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | - | Field label |
| `hint` | `string` | - | Helper text below input |
| `error` | `string` | - | Error message |
| `required` | `boolean` | `false` | Show required indicator |
| `children` | `ReactNode` | - | Input element |

## Design Notes

- Uses **Flat** archetype
- Label positioned above input
- Error message replaces hint when present
- Required indicator uses error color
- Consistent spacing with 4px gap

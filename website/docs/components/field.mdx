# Field

Label and validation wrapper for form inputs.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | - | Field label |
| `hint` | `string` | - | Helper text |
| `error` | `string` | - | Error message |
| `required` | `boolean` | `false` | Show required indicator |
| `layout` | `'vertical'` \| `'horizontal'` | `'vertical'` | Label position |
| `children` | `ReactNode` | - | Input element |

## Accessibility

- Associates label with input via `htmlFor`
- Error message linked via `aria-describedby`
- Required field indicated visually and via `aria-required`

## Usage with Validation

```tsx
import { useState } from 'react'
import { Field, Input } from 'intent-react'

function ValidatedField() {
  const [value, setValue] = useState('')
  const [error, setError] = useState('')
  
  const validate = (val) => {
    if (val.length < 3) {
      setError('Must be at least 3 characters')
    } else {
      setError('')
    }
  }
  
  return (
    <Field label="Username" error={error} required>
      <Input 
        value={value}
        onChange={(e) => {
          setValue(e.target.value)
          validate(e.target.value)
        }}
        validation={error ? 'invalid' : 'none'}
      />
    </Field>
  )
}
```

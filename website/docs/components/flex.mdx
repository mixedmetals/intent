# Flex

Flexible box container with alignment options.

## Examples

### Basic Flex

```tsx
import { Flex, Card, Text } from 'intent-react'

<Flex gap="normal">
  <Card padding="normal"><Text>Item 1</Text></Card>
  <Card padding="normal"><Text>Item 2</Text></Card>
  <Card padding="normal"><Text>Item 3</Text></Card>
</Flex>
```

### Justify Content

```tsx
<Flex justify="between" gap="normal">
  <Card padding="normal"><Text>Left</Text></Card>
  <Card padding="normal"><Text>Center</Text></Card>
  <Card padding="normal"><Text>Right</Text></Card>
</Flex>
```

### Align Items

```tsx
<Flex align="center" gap="normal">
  <Card padding="compact">Small</Card>
  <Card padding="normal">Medium</Card>
  <Card padding="relaxed">Large</Card>
</Flex>
```

### Wrap

```tsx
<Flex wrap gap="normal">
  <Card padding="normal" style={{ width: 150 }}>Item 1</Card>
  <Card padding="normal" style={{ width: 150 }}>Item 2</Card>
  <Card padding="normal" style={{ width: 150 }}>Item 3</Card>
  <Card padding="normal" style={{ width: 150 }}>Item 4</Card>
  <Card padding="normal" style={{ width: 150 }}>Item 5</Card>
</Flex>
```

### Column Direction

```tsx
<Flex direction="column" gap="normal">
  <Card padding="normal">First</Card>
  <Card padding="normal">Second</Card>
  <Card padding="normal">Third</Card>
</Flex>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `direction` | `'row'` \| `'column'` | `'row'` | Flex direction |
| `gap` | `'compact'` \| `'normal'` \| `'relaxed'` \| `'loose'` | `'normal'` | Gap between items |
| `justify` | `'start'` \| `'center'` \| `'end'` \| `'between'` \| `'around'` \| `'evenly'` | `'start'` | Justify content |
| `align` | `'start'` \| `'center'` \| `'end'` \| `'stretch'` \| `'baseline'` | `'stretch'` | Align items |
| `wrap` | `boolean` \| `'wrap'` \| `'nowrap'` \| `'wrap-reverse'` | `'nowrap'` | Flex wrap |
| `inline` | `boolean` | `false` | Use inline-flex |

## CSS Output

```css
.flex {
  display: flex;
}

.flex--inline {
  display: inline-flex;
}

.flex--direction-column {
  flex-direction: column;
}

.flex--gap-normal {
  gap: var(--intent-spacing-normal);
}

.flex--justify-between {
  justify-content: space-between;
}

.flex--align-center {
  align-items: center;
}

.flex--wrap {
  flex-wrap: wrap;
}
```

## Flex vs Stack

Use **Flex** when:
- You need full flexbox control (justify, align, wrap)
- More complex layouts
- Alignment is important

Use **Stack** when:
- Simple one-dimensional spacing
- Consistent gap sizes
- Quick layouts without thinking about flexbox

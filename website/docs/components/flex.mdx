# Flex

Flexible box container with alignment options.

:::note
Use `Stack` (VStack/HStack) for most layouts. Use `Box` with flex styles for advanced flex configurations.
:::

## Examples

### Basic Flex with Box

<div className="intent-demo">
  <Box display="flex" gap="normal">
    <Box p="normal" style={{ background: 'var(--intent-color-surface-elevated)' }}>Item 1</Box>
    <Box p="normal" style={{ background: 'var(--intent-color-surface-elevated)' }}>Item 2</Box>
    <Box p="normal" style={{ background: 'var(--intent-color-surface-elevated)' }}>Item 3</Box>
  </Box>
</div>

```tsx
import { Box } from 'intent-react/components/layout';

<Box display="flex" gap="normal">
  <Box p="normal">Item 1</Box>
  <Box p="normal">Item 2</Box>
  <Box p="normal">Item 3</Box>
</Box>
```

### Justify Content

<div className="intent-demo">
  <Box display="flex" gap="normal" justifyContent="space-between">
    <Box p="normal" style={{ background: 'var(--intent-color-surface-elevated)' }}>Left</Box>
    <Box p="normal" style={{ background: 'var(--intent-color-surface-elevated)' }}>Right</Box>
  </Box>
</div>

```tsx
<Box display="flex" justifyContent="space-between">...</Box>
<Box display="flex" justifyContent="center">...</Box>
<Box display="flex" justifyContent="flex-end">...</Box>
```

### Align Items

<div className="intent-demo">
  <Box display="flex" gap="normal" alignItems="center" style={{ height: '100px' }}>
    <Box p="normal" style={{ background: 'var(--intent-color-surface-elevated)' }}>Center</Box>
    <Box p="normal" style={{ background: 'var(--intent-color-surface-elevated)' }}>Aligned</Box>
  </Box>
</div>

```tsx
<Box display="flex" alignItems="center">...</Box>
<Box display="flex" alignItems="flex-start">...</Box>
<Box display="flex" alignItems="flex-end">...</Box>
```

## Box Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `display` | `'flex' \| 'block' \| 'inline' \| 'none'` | `'block'` | Display property |
| `gap` | `'compact' \| 'normal' \| 'relaxed' \| 'loose'` | - | Flex gap |
| `justifyContent` | `string` | - | Justify content |
| `alignItems` | `string` | - | Align items |
| `flexDirection` | `'row' \| 'column'` | - | Flex direction |
| `flexWrap` | `'wrap' \| 'nowrap'` | - | Flex wrap |
| `p` | `'compact' \| 'normal' \| 'relaxed' \| 'loose'` | - | Padding |
| `m` | `'compact' \| 'normal' \| 'relaxed' \| 'loose'` | - | Margin |

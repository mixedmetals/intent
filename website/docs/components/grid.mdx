# Grid

Two-dimensional layout system for complex arrangements.

## Examples

### Basic Grid

<div className="intent-demo">

    <GridItem><Card>Item 4</Card></GridItem>
    <GridItem><Card>Item 5</Card></GridItem>
    <GridItem><Card>Item 6</Card></GridItem>
  </Grid>


## Props

### Grid

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `columns` | `number \| object` | - | Number of columns or responsive config |
| `rows` | `number \| object` | - | Number of rows |
| `gap` | `'compact' \| 'normal' \| 'relaxed' \| 'loose'` | `'normal'` | Gap between items |
| `flow` | `'row' \| 'column' \| 'dense'` | `'row'` | Grid auto-flow |

### GridItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `colSpan` | `number` | - | Columns to span |
| `rowSpan` | `number` | - | Rows to span |
| `colStart` | `number` | - | Start column |
| `colEnd` | `number` | - | End column |

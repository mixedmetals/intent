# Grid

Two-dimensional layout system for complex arrangements.

## Examples

### Basic Grid

<Grid columns={3} gap="normal" style={{ marginBottom: 24 }}>
  <Card padding="normal"><Text>Item 1</Text></Card>
  <Card padding="normal"><Text>Item 2</Text></Card>
  <Card padding="normal"><Text>Item 3</Text></Card>
  <Card padding="normal"><Text>Item 4</Text></Card>
  <Card padding="normal"><Text>Item 5</Text></Card>
  <Card padding="normal"><Text>Item 6</Text></Card>
</Grid>

```tsx
<Grid columns={3} gap="normal">
  <Card padding="normal"><Text>Item 1</Text></Card>
  <Card padding="normal"><Text>Item 2</Text></Card>
  ...
</Grid>
```

### Column Span

<Grid columns={3} gap="normal" style={{ marginBottom: 24 }}>
  <Card padding="normal" style={{ gridColumn: 'span 2' }}>
    <Text>Span 2 columns</Text>
  </Card>
  <Card padding="normal"><Text>Item</Text></Card>
  <Card padding="normal"><Text>Item</Text></Card>
  <Card padding="normal" style={{ gridColumn: 'span 3' }}>
    <Text>Span all 3 columns</Text>
  </Card>
</Grid>

```tsx
<Grid columns={3} gap="normal">
  <Card style={{ gridColumn: 'span 2' }}>...</Card>
  <Card>...</Card>
  <Card style={{ gridColumn: 'span 3' }}>...</Card>
</Grid>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `columns` | `number` \| `object` \| `'auto-fit'` | `3` | Number of columns |
| `gap` | `'compact'` \| `'normal'` \| `'relaxed'` \| `'loose'` | `'normal'` | Grid gap |
| `minColumnWidth` | `number` | `250` | Min width for auto-fit |
| `alignItems` | `'start'` \| `'center'` \| `'end'` \| `'stretch'` | `'stretch'` | Vertical alignment |
| `justifyItems` | `'start'` \| `'center'` \| `'end'` \| `'stretch'` | `'stretch'` | Horizontal alignment |

## Responsive Columns Object

```tsx
<Grid columns={{ 
  sm: 1,   // < 640px
  md: 2,   // 640px - 768px
  lg: 3,   // 768px - 1024px
  xl: 4    // > 1024px
}}>
  ...
</Grid>
```

## CSS Output

```css
.grid {
  display: grid;
}

.grid--columns-3 {
  grid-template-columns: repeat(3, 1fr);
}

.grid--columns-auto-fit {
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
}

.grid--gap-normal {
  gap: var(--intent-spacing-normal);
}

@media (max-width: 640px) {
  .grid--columns-sm-1 {
    grid-template-columns: repeat(1, 1fr);
  }
}
```

## Grid vs Stack

Use **Grid** when:
- You need two-dimensional layout
- Items should align in rows and columns
- You want responsive column counts

Use **Stack** when:
- One-dimensional layout (row or column)
- Simple spacing between items
- Easier composition

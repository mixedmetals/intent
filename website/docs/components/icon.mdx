# Icon

Display icon from the icon library.

## Example

```tsx
import { Icon } from 'intent-react'

<Icon name="home" />
<Icon name="user" />
<Icon name="settings" />
```

### Sizes

```tsx
<Icon name="star" size={12} />
<Icon name="star" size={16} />
<Icon name="star" size={24} />
<Icon name="star" size={32} />
```

### Colors

```tsx
<Icon name="circle" color="default" />
<Icon name="circle" color="primary" />
<Icon name="circle" color="success" />
<Icon name="circle" color="warning" />
<Icon name="circle" color="danger" />
```

### With Button

```tsx
<Button icon="save">Save</Button>
<Button icon="trash" importance="tertiary">Delete</Button>
<Button icon="arrow-right" iconPosition="right">Next</Button>
```

### Rotating

```tsx
<Icon name="loader" spin />
<Icon name="refresh-cw" spin />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `name` | `string` | required | Icon name |
| `size` | `number` | `20` | Icon size in pixels |
| `color` | `'default'` \| `'primary'` \| `'success'` \| `'warning'` \| `'danger'` \| `'muted'` | `'default'` | Icon color |
| `spin` | `boolean` | `false` | Rotate animation |
| `strokeWidth` | `number` | `2` | Stroke width |

## CSS Output

```css
.icon {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  flex-shrink: 0;
}

.icon--spin {
  animation: icon-spin 1s linear infinite;
}

@keyframes icon-spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}

.icon--color-primary {
  color: var(--intent-color-primary);
}
```

## Available Icons

Intent includes a comprehensive icon set covering:
- Interface icons (home, settings, user, etc.)
- Navigation icons (arrow, chevron, menu, etc.)
- Action icons (edit, delete, save, etc.)
- Status icons (check, alert, info, etc.)
- File icons (file, folder, image, etc.)

Icons are imported from the [Lucide](https://lucide.dev/) icon library.

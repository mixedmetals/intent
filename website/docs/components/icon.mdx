# Icon

## Props

### Icon

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `name` | `string` | - | Icon name from registry |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Icon size |
| `color` | `string` | - | Icon color |

### IconText

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `name` | `string` | - | Icon name |
| `text` | `string` | - | Text to display |
| `alignment` | `'left' \| 'right'` | `'left'` | Icon position |

## Available Icons

Intent includes a comprehensive icon registry:

```tsx
import { iconRegistry, getIconNames } from 'intent-react';

// Get all available icon names
const icons = getIconNames();

// Check if icon exists
import { hasIcon } from 'intent-react';
const exists = hasIcon('arrow-right');
```

## Design Notes

- Uses SVG for crisp rendering at any size
- Consistent sizing scale (xs to xl)
- Color inherits from parent by default
- IconText handles spacing between icon and text

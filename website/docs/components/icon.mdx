# Icon

Display icon from the icon library.

## Example

```tsx
import { Icon } from 'intent-react'

<Icon name="home" />
<Icon name="user" />
<Icon name="settings" />
```

### Sizes

```tsx
<Icon name="star" size={12} />
<Icon name="star" size={24} />
<Icon name="star" size={32} />
```

### Colors

```tsx
<Icon name="circle" color="primary" />
<Icon name="circle" color="success" />
<Icon name="circle" color="danger" />
```

### Rotating

```tsx
<Icon name="loader" spin />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `name` | `string` | required | Icon name |
| `size` | `number` | `20` | Icon size |
| `color` | `string` | `'default'` | Icon color |
| `spin` | `boolean` | `false` | Rotate animation |

## CSS Output

```css
.icon {
  display: inline-flex;
  align-items: center;
  justify-content: center;
}

.icon--spin {
  animation: icon-spin 1s linear infinite;
}

@keyframes icon-spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
```

## Available Icons

Intent uses the [Lucide](https://lucide.dev/) icon library.

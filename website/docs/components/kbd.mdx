# Kbd

Keyboard key indicator for shortcuts.

## Example

```tsx
import { Kbd, Text } from 'intent-react'

<Text>
  Press <Kbd>Cmd</Kbd> + <Kbd>K</Kbd> to open command palette
</Text>
```

### Common Shortcuts

```tsx
<Text><Kbd>Ctrl</Kbd> + <Kbd>C</Kbd> to copy</Text>
<Text><Kbd>Ctrl</Kbd> + <Kbd>V</Kbd> to paste</Text>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `string` | required | Key label |
| `size` | `'sm'` \| `'md'` | `'md'` | Key size |

# Kbd

Keyboard key component for shortcuts. Uses the **Milled** archetype for tactile key appearance.

## Examples

### Basic Keys

<div className="intent-demo">
  <div style={{ display: 'flex', gap: '8px', alignItems: 'center' }}>
    <Kbd>Ctrl</Kbd>
    <span>+</span>
    <Kbd>C</Kbd>
  </div>
</div>

```tsx
import { Kbd } from 'intent-react';

<Kbd>Ctrl</Kbd> + <Kbd>C</Kbd>
```

### Sizes

```tsx
<Kbd size="sm">Small</Kbd>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|

# Label

Accessible form labels. Uses the **Flat** archetype with subtle styling.

## Examples

### Basic Label

<div className="intent-demo">
  <Label htmlFor="email">Email Address</Label>
  <input 
    id="email" 
    type="email" 
    placeholder="Enter your email"
    style={{ 
      marginTop: '4px', 
      width: '100%', 
      padding: '8px 12px', 
      border: '1px solid var(--intent-color-border)', 
      borderRadius: '6px' 
    }} 
  />
</div>

```tsx
import { Label, Input } from 'intent-react';

<Label htmlFor="email">Email Address</Label>
<Input id="email" type="email" />
```

### Required Field

<div className="intent-demo">

    }} 
  />
</div>

```tsx
<Label htmlFor="name" required>Name</Label>

```

### Disabled State

<div className="intent-demo">
  <Label htmlFor="disabled" disabled>Disabled Label</Label>
  <input 
    id="disabled" 
    type="text" 
    disabled
    placeholder="Cannot edit"
    style={{ 
      marginTop: '4px', 
      width: '100%', 
      padding: '8px 12px', 
      border: '1px solid var(--intent-color-border)', 
      borderRadius: '6px',
      opacity: 0.5
    }} 
  />
</div>

```tsx
<Label htmlFor="disabled" disabled>Disabled Label</Label>
<Input id="disabled" disabled />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `htmlFor` | `string` | - | Associated input ID |
| `required` | `boolean` | `false` | Show required indicator |
| `disabled` | `boolean` | `false` | Disabled state styling |

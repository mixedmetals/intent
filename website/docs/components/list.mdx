# List

Display lists of items with various styles. Uses the **Flat** archetype.

## Examples

### Basic List

<div className="intent-demo">
  <List>
    <ListItem>User Settings</ListItem>
    <ListItem>Notifications</ListItem>
    <ListItem>Privacy</ListItem>
    <ListItem>Security</ListItem>
  </List>
</div>

```tsx
import { List, ListItem } from 'intent-react';

<List>
  <ListItem>User Settings</ListItem>
  <ListItem>Notifications</ListItem>
</List>
```

## Props

### List

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `'unordered' \| 'ordered'` | `'unordered'` | List type |
| `spacing` | `'compact' \| 'normal' \| 'relaxed'` | `'normal'` | Item spacing |

### ListItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `icon` | `ReactNode` | - | Leading icon |
| `action` | `ReactNode` | - | Trailing action |

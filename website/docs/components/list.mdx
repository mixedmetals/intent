# List

Vertical list of items with various styles.

## Example

<List 
  items={[
    { id: 1, title: 'Inbox', badge: 12, icon: 'inbox' },
    { id: 2, title: 'Starred', icon: 'star' },
    { id: 3, title: 'Sent', icon: 'send' },
    { id: 4, title: 'Drafts', badge: 3, icon: 'file' },
    { id: 5, title: 'Trash', icon: 'trash' },
  ]}
  style={{ marginBottom: 24 }}
/>

## Selectable

```tsx
const [selectedId, setSelectedId] = useState(null)

<List 
  selectable
  selectedId={selectedId}
  onSelect={setSelectedId}
  items={[
    { id: 1, title: 'Option 1' },
    { id: 2, title: 'Option 2' },
    { id: 3, title: 'Option 3' },
  ]}
/>
```

## With Actions

```tsx
<List 
  items={[
    { 
      id: 1, 
      title: 'Document.pdf', 
      subtitle: '2.5 MB',
      icon: 'file-text',
      actions: [
        { icon: 'download', onClick: () => download(1) },
        { icon: 'trash', danger: true, onClick: () => delete(1) },
      ]
    },
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `ListItem[]` | required | List items |
| `variant` | `'default'` \| `'bordered'` \| `'divided'` | `'default'` | List style |
| `selectable` | `boolean` | `false` | Allow selection |
| `selectedId` | `string` \| `number` | - | Selected item ID |
| `onSelect` | `(id: string \| number) => void` | - | Select callback |
| `emptyMessage` | `string` | `'No items'` | Empty state message |

### ListItem Type

```typescript
interface ListItem {
  id: string | number
  title: string
  subtitle?: string
  icon?: string
  avatar?: string
  badge?: string | number
  actions?: Action[]
  disabled?: boolean
}

interface Action {
  icon: string
  onClick: () => void
  danger?: boolean
}
```

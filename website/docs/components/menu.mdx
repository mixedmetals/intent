# Menu

Dropdown menus for actions and navigation. Uses the **Elevated** archetype.

:::tip
For more complex menus with submenus, use `DropdownMenu`.
:::

## Props

### Menu

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `trigger` | `ReactNode` | - | Menu trigger element |
| `children` | `ReactNode` | - | Menu items |

### MenuItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `icon` | `ReactNode` | - | Optional icon |
| `destructive` | `boolean` | `false` | Error color styling |
| `onClick` | `() => void` | - | Click handler |
| `children` | `ReactNode` | - | Item label |

### MenuDivider

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| - | - | - | Visual separator |

## Design Notes

- Menu uses **Elevated** archetype
- Items use **Flat** archetype with hover state
- Divider uses 1px border color
- Destructive items use error color
- Icons are optional but recommended

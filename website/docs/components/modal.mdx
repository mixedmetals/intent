# Modal

Overlay dialogs for important actions. Uses the **Elevated** archetype for the dialog panel.

## Examples

### Basic Modal

<div className="intent-demo">
  <ModalDemo />
</div>

```tsx
import { Modal } from 'intent-react';

const [isOpen, setIsOpen] = useState(false)

<Button onClick={() => setIsOpen(true)}>Open Modal</Button>

<Modal 
  isOpen={isOpen} 
  onClose={() => setIsOpen(false)}
  title="Confirm Action"
  footer={
    <>
      <Button importance="secondary" onClick={() => setIsOpen(false)}>Cancel</Button>
      <Button onClick={() => setIsOpen(false)}>Confirm</Button>
    </>
  }
>
  Are you sure you want to continue?
</Modal>
```

### Sizes

```tsx
<Modal size="sm">Small confirmation</Modal>
<Modal size="md">Standard dialog</Modal>
<Modal size="lg">Large content</Modal>
<Modal size="xl">Full-featured dialog</Modal>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `isOpen` | `boolean` | `false` | Whether modal is open |
| `onClose` | `() => void` | - | Called when modal closes |
| `title` | `string` | - | Modal title |
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl' \| 'full'` | `'md'` | Modal size |
| `children` | `ReactNode` | - | Modal content |
| `footer` | `ReactNode` | - | Footer content |

## Design Notes

- Modal panel uses **Elevated** archetype (floating above content)
- Backdrop uses `rgba(0,0,0,0.5)` with `backdrop-filter: blur(4px)`
- Larger shadow than Card to indicate higher elevation
- Header and footer have subtle 1px borders

## Props

### Navbar

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `position` | `'static' \| 'fixed' \| 'sticky'` | `'static'` | Position behavior |
| `border` | `boolean` | `true` | Show bottom border |

### NavbarItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `active` | `boolean` | `false` | Active state |
| `href` | `string` | - | Link URL |

## Design Notes

- Uses **Elevated** archetype (slight elevation)
- Bottom border for separation
- Active item uses primary color
- Supports brand logo, navigation items, and actions
- Responsive behavior with mobile menu

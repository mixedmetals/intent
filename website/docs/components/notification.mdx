# Notification

Toast notifications and alerts. Uses the **Elevated** archetype.

:::tip
Use the `useToast` hook for displaying notifications programmatically.
:::

## useToast Hook

```tsx
import { useToast, Button } from 'intent-react';

function Example() {
  const toast = useToast();
  
  return (
    <Button onClick={() => toast({ title: "Hello!", status: "info" })}>
      Show Toast
    </Button>
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | - | Notification title |
| `description` | `string` | - | Notification message |
| `status` | `'info' \| 'success' \| 'warning' \| 'error'` | `'info'` | Notification style |
| `duration` | `number` | `5000` | Auto-dismiss duration (ms) |
| `isClosable` | `boolean` | `true` | Show close button |
| `position` | `'top' \| 'bottom' \| 'top-right' \| ...` | `'bottom'` | Position on screen |

## Design Notes

- Uses **Elevated** archetype
- Status icons use semantic colors
- Close button on the right
- Auto-dismisses after duration
- Supports multiple positions

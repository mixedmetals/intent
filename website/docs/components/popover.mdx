# Popover

Floating content panel triggered by clicking an element.

## Example

<PopoverDemo />

```tsx
import { useState } from 'react'
import { Button, Popover, Stack, Field, Input } from 'intent-react'

function FilterPopover() {
  const [open, setOpen] = useState(false)
  
  return (
    <Popover 
      open={open}
      onOpenChange={setOpen}
      trigger={<Button>Filter</Button>}
    >
      <Stack gap="normal" style={{ width: 240 }}>
        <Field label="Status">
          <Select options={[...]} />
        </Field>
        <Stack direction="row" gap="normal">
          <Button onClick={() => setOpen(false)}>Apply</Button>
        </Stack>
      </Stack>
    </Popover>
  )
}
```

## With Form

```tsx
<Popover trigger={<Button>Add User</Button>}>
  <form onSubmit={handleSubmit}>
    <Stack gap="normal">
      <Field label="Name">
        <Input placeholder="Full name" />
      </Field>
      <Field label="Email">
        <Input type="email" placeholder="Email address" />
      </Field>
      <Button type="submit" importance="primary" fullWidth>
        Create User
      </Button>
    </Stack>
  </form>
</Popover>
```

## Alignment

```tsx
<Popover align="start" trigger={...}>...</Popover>
<Popover align="center" trigger={...}>...</Popover>
<Popover align="end" trigger={...}>...</Popover>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `trigger` | `ReactNode` | required | Element that opens the popover |
| `children` | `ReactNode` | - | Popover content |
| `open` | `boolean` | - | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | - | Open state callback |
| `side` | `'top'` \| `'right'` \| `'bottom'` \| `'left'` | `'bottom'` | Popover position |
| `align` | `'start'` \| `'center'` \| `'end'` | `'center'` | Content alignment |
| `modal` | `boolean` | `false` | Trap focus inside |

## Popover vs Tooltip vs Dropdown

Use **Popover** for:
- Interactive content (forms, buttons)
- Rich content (images, formatted text)
- Confirmations with context

Use **Tooltip** for:
- Simple text hints
- Icon labels
- Brief explanations

Use **Dropdown** for:
- Action menus
- Navigation options
- Selection from a list

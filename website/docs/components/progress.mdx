# Progress

Visual indicator for completion percentage. Uses the **Flat** archetype for the track and fill.

  </VStack>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | `0` | Current value (0-100) |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Progress bar height |
| `label` | `string` | - | Label text |
| `indeterminate` | `boolean` | `false` | Indeterminate animation |

## Design Notes

- Track uses `var(--intent-color-surface-elevated)` for subtle background
- Fill uses `var(--intent-color-primary)` for brand consistency
- Smooth width transition on value change (150ms ease)
- Border radius is fully rounded (`9999px`)

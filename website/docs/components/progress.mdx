## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | `0` | Current value |
| `max` | `number` | `100` | Maximum value |
| `size` | `'small'` \| `'medium'` \| `'large'` | `'medium'` | Progress height |
| `variant` | `'default'` \| `'success'` \| `'warning'` \| `'danger'` | `'default'` | Color variant |
| `indeterminate` | `boolean` | `false` | Loading animation |
| `label` | `string` | - | Text label |
| `showLabel` | `boolean` | `true` | Show percentage label |

## Accessibility

- Uses `role="progressbar"`
- Includes `aria-valuenow`, `aria-valuemin`, `aria-valuemax`
- Label announced by screen readers

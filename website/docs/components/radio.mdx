# Radio

Single-selection from a group of options.

export const Radio = ({ label, description, name, value, checked, disabled }) => (
  <label className="intent-radio">
    <input 
      type="radio"
      name={name}
      value={value}
      className="intent-radio__input"
      checked={checked}
      disabled={disabled}
    />
    <div>
      <div className="intent-radio__label">{label}</div>
      {description && <div style={{ fontSize: '14px', color: 'var(--intent-color-text-muted)', marginTop: '2px' }}>{description}</div>}
    </div>
  </label>
)

export const Stack = ({ children, gap = 'normal' }) => (
  <div className={`intent-stack intent-stack--direction-column intent-stack--gap-${gap}`}>
    {children}
  </div>
)

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `name` | `string` | - | Radio group name |
| `value` | `string` | - | Radio value |
| `label` | `string` | - | Radio label |
| `description` | `string` | - | Helper text |
| `checked` | `boolean` | - | Controlled checked state |
| `disabled` | `boolean` | `false` | Disable radio |
| `onChange` | `(e: ChangeEvent) => void` | - | Change handler |

## Radio vs Checkbox

Use **Radio** when:
- Only one option can be selected
- Selecting an option immediately shows the result
- All options should be visible

Use **Checkbox** when:
- Multiple options can be selected
- Options are independent
- Toggle a single on/off state

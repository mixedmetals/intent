# Separator

Visual dividers between content. Uses the **Flat** archetype with 1px borders.

## Examples

### Horizontal

<div className="intent-demo">
  <div>Content above</div>
  <div style={{ height: '1px', width: '100%', background: 'var(--intent-color-border)', margin: '16px 0' }} />
  <div>Content below</div>
</div>

```tsx
import { Separator } from 'intent-react';

<div>Content above</div>
<Separator />
<div>Content below</div>
```

### With Label

<div className="intent-demo">
  <div style={{ display: 'flex', flexDirection: 'column', gap: '12px' }}>
    <button className="intent-button intent-button--importance-primary intent-button--size-medium" style={{ width: '100%' }}>
      Sign in with Email
    </button>
    <div style={{ display: 'flex', alignItems: 'center', gap: '16px', width: '100%' }}>
      <div style={{ flex: 1, height: '1px', background: 'var(--intent-color-border)' }} />
      <span style={{ fontSize: '0.875rem', color: 'var(--intent-color-muted)', fontWeight: 500 }}>OR</span>
      <div style={{ flex: 1, height: '1px', background: 'var(--intent-color-border)' }} />
    </div>
    <button className="intent-button intent-button--importance-secondary intent-button--size-medium" style={{ width: '100%' }}>
      Sign in with Google
    </button>
  </div>
</div>

```tsx
<Button>Sign in with Email</Button>
<Separator label="OR" />
<Button importance="secondary">Sign in with Google</Button>
```

### Vertical

<div className="intent-demo">
  <div style={{ display: 'flex', alignItems: 'center', gap: '16px', height: '40px' }}>
    <span>Left content</span>
    <div style={{ width: '1px', height: '100%', background: 'var(--intent-color-border)' }} />
    <span>Right content</span>
  </div>
</div>

```tsx
<div style={{ display: 'flex', alignItems: 'center', gap: '16px' }}>
  <span>Left</span>
  <Separator orientation="vertical" />
  <span>Right</span>
</div>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Separator direction |
| `label` | `string` | - | Center label text |
| `decorative` | `boolean` | `false` | If true, hidden from screen readers |

## Design Notes

- Uses **Flat** archetype (no bevel)
- 1px border using `var(--intent-color-border)`
- Label uses muted text color
- Vertical separator has minimum height

# Sidebar

Application sidebar navigation. Uses the **Elevated** archetype for the container.

## Props

### Sidebar

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `collapsible` | `boolean` | `false` | Enable collapse behavior |
| `defaultCollapsed` | `boolean` | `false` | Initial collapsed state |
| `width` | `string` | `'240px'` | Sidebar width |

### SidebarItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `icon` | `ReactNode` | - | Item icon |
| `active` | `boolean` | `false` | Active state |
| `badge` | `number \| string` | - | Badge content |
| `href` | `string` | - | Link URL |

### SidebarSection

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | - | Section title |
| `collapsible` | `boolean` | `false` | Allow section collapse |

## Design Notes

- Uses **Elevated** archetype for background
- Active item uses primary color with subtle background
- Icons are optional but recommended
- Badges aligned to the right
- Collapsible on mobile

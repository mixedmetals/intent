# Sidebar

Vertical navigation for application side menu.

## Example

```tsx
import { Sidebar, Badge } from 'intent-react'

<Sidebar
  logo={<Logo />}
  items={[
    { 
      label: 'Dashboard', 
      icon: 'layout-dashboard', 
      href: '/dashboard',
      active: true 
    },
    { 
      label: 'Projects', 
      icon: 'folder',
      href: '/projects',
      badge: '12'
    },
    { 
      label: 'Tasks', 
      icon: 'check-square',
      href: '/tasks'
    },
    { 
      label: 'Team', 
      icon: 'users',
      href: '/team'
    },
    { 
      label: 'Settings', 
      icon: 'settings',
      href: '/settings'
    },
  ]}
/>
```

## Collapsible

```tsx
const [collapsed, setCollapsed] = useState(false)

<Sidebar
  collapsible
  collapsed={collapsed}
  onCollapse={setCollapsed}
  logo={<Logo />}
  items={[...]}
/>
```


## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `logo` | `ReactNode` | - | Brand logo |
| `items` | `SidebarItem[]` | `[]` | Navigation items |
| `sections` | `Section[]` | - | Grouped sections |
| `collapsible` | `boolean` | `false` | Allow collapse |
| `collapsed` | `boolean` | `false` | Collapsed state |
| `onCollapse` | `(collapsed: boolean) => void` | - | Collapse callback |
| `footer` | `ReactNode` | - | Footer content |
| `width` | `number` | `240` | Sidebar width (px) |
| `collapsedWidth` | `number` | `64` | Collapsed width (px) |

### SidebarItem Type

```typescript
interface SidebarItem {
  label: string
  icon?: string
  href?: string
  onClick?: () => void
  active?: boolean
  disabled?: boolean
  badge?: string | number
  items?: SidebarItem[]  // Nested items
}
```

### Section Type

```typescript
interface Section {
  title?: string
  items: SidebarItem[]
}
```

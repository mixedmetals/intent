# Skeleton

Loading placeholder for content that hasn't loaded yet. Uses the **Flat** archetype with subtle animation.

## Examples

    <Skeleton width="md" height="sm" />
  </div>
</div>

## Props

### Skeleton

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'text' \| 'circle' \| 'rectangle'` | `'text'` | Skeleton shape |
| `width` | `'auto' \| 'full' \| 'sm' \| 'md' \| 'lg'` | `'auto'` | Width preset |
| `height` | `'auto' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'auto'` | Height preset |

### SkeletonText

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `lines` | `number` | `3` | Number of lines |
| `lastLineWidth` | `'full' \| 'sm' \| 'md' \| 'lg'` | `'md'` | Width of last line |

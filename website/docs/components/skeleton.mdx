## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `height` | `number` \| `string` | - | Skeleton height |
| `width` | `number` \| `string` | `'100%'` | Skeleton width |
| `circle` | `boolean` | `false` | Circular shape |
| `animate` | `boolean` | `true` | Pulse animation |
| `style` | `CSSProperties` | - | Additional styles |

### Skeleton.Text Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `lines` | `number` | `3` | Number of lines |
| `lineHeight` | `number` | 16 | Height of each line |
| `lastLineWidth` | `string` | `'80%'` | Width of last line |

## Usage Pattern

```tsx
import { useState, useEffect } from 'react'
import { Card, Skeleton, Text } from 'intent-react'

function UserProfile({ userId }) {
  const [user, setUser] = useState(null)
  const [loading, setLoading] = useState(true)
  
  useEffect(() => {
    fetchUser(userId).then(data => {
      setUser(data)
      setLoading(false)
    })
  }, [userId])
  
  if (loading) {
    return (
      <Card padding="normal">
        <Skeleton circle width={64} height={64} />
        <Skeleton height={24} width="60%" />
        <Skeleton.Text lines={2} />
      </Card>
    )
  }
  
  return (
    <Card padding="normal">
      <Avatar src={user.avatar} size="lg" />
      <Text weight="bold">{user.name}</Text>
      <Text color="muted">{user.bio}</Text>
    </Card>
  )
}
```

```css
@keyframes skeleton-pulse {

  }

  }
}

.skeleton--circle {

}
```

## When to Use

Use Skeleton when:
- Content is loading and will appear in the same layout
- You want to reduce perceived loading time
- The layout is predictable

Use Spinner when:
- Loading time is unknown or very short
- You don't know the final layout
- For full-page loading states

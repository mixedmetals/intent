# Spacer

Flexible spacing component for layouts.

## Examples

### Spacer Sizes

<div className="intent-demo">
  <div>Content above</div>
  <div style={{ height: '16px', background: 'repeating-linear-gradient(45deg, transparent, transparent 5px, var(--intent-color-border) 5px, var(--intent-color-border) 6px)', borderRadius: '4px', opacity: 0.3, margin: '8px 0' }} />
  <div>Content below (16px gap)</div>
  <div style={{ height: '32px', background: 'repeating-linear-gradient(45deg, transparent, transparent 5px, var(--intent-color-border) 5px, var(--intent-color-border) 6px)', borderRadius: '4px', opacity: 0.3, margin: '8px 0' }} />
  <div>More content (32px gap)</div>
</div>

```tsx
import { Spacer } from 'intent-react/components/layout';

<div>Content A</div>
<Spacer size={4} />
<div>Content B</div>
<Spacer size={8} />
<div>Content C</div>
```

### Horizontal Spacer

<div className="intent-demo">
  <div style={{ display: 'flex', alignItems: 'center' }}>
    <span>Item 1</span>
    <div style={{ width: '16px' }} />
    <span>Item 2</span>
    <div style={{ flex: 1 }} />
    <span>Item 3 (pushed right)</span>
  </div>
</div>

```tsx
<Flex>
  <Text>Left</Text>
  <Spacer flex={1} />
  <Text>Right</Text>
</Flex>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `number` | `4` | Spacer size (1-16, multiples of 4px) |
| `flex` | `number` | - | Flex grow value for horizontal spacing |

## Size Scale

| Size | Height |
|------|--------|
| 1 | 4px |
| 2 | 8px |
| 4 | 16px |
| 6 | 24px |
| 8 | 32px |
| 12 | 48px |
| 16 | 64px |

## Design Notes

- Uses **Flat** archetype (no bevel)
- Size follows 4px grid system
- Use `flex` prop for horizontal spacing in flex containers
- Invisible by default (no background)

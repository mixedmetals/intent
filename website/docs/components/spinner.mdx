import { Button } from 'intent-react';
import { HStack } from 'intent-react';

# Spinner

Loading indicator with multiple sizes. Uses the **Flat** archetype (no bevel).

## Examples

### Sizes

<div className="intent-demo">
  <HStack gap="normal" align="center">
    <Spinner size="sm" />
    <Spinner size="md" />
    <Spinner size="lg" />
  </HStack>
</div>

```tsx
import { Spinner } from 'intent-react';

<Spinner size="sm" />
<Spinner size="md" />
<Spinner size="lg" />
```

### Inside Button

<div className="intent-demo">
  <HStack gap="normal">
    <Button importance="secondary" disabled>
      <Spinner size="sm" />
      <span style={{ marginLeft: 8 }}>Loading...</span>
    </Button>
  </HStack>
</div>

```tsx
<Button disabled>
  <Spinner size="sm" />
  Loading...
</Button>
```

### Colors

<div className="intent-demo">
  <HStack gap="normal" align="center">
    <Spinner color="primary" />
    <Spinner color="secondary" />
    <div style={{ background: 'var(--intent-color-primary)', padding: '8px', borderRadius: '4px' }}>
      <Spinner color="white" />
    </div>
  </HStack>
</div>

```tsx
<Spinner color="primary" />
<Spinner color="secondary" />
<Spinner color="white" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Spinner size |
| `color` | `'primary' \| 'secondary' \| 'white'` | `'primary'` | Spinner color |

# Stack

The foundation of Intent layouts. Stack provides vertical and horizontal spacing with consistent, theme-aware gaps.

## Overview

Stack is the most fundamental layout component in Intent. It handles spacing between child elements consistently across your application.

## Examples

### Vertical Stack (Default)

<div className="intent-demo">
  <VStack gap="normal">
    <Card>Item 1</Card>
    <Card>Item 2</Card>
    <Card>Item 3</Card>
  </VStack>
</div>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `direction` | `'column' \| 'row'` | `'column'` | Stack direction |
| `gap` | `'compact' \| 'normal' \| 'relaxed' \| 'loose'` | `'normal'` | Gap between items |
| `align` | `'start' \| 'center' \| 'end' \| 'stretch'` | - | Cross-axis alignment |
| `justify` | `'start' \| 'center' \| 'end' \| 'between' \| 'around'` | - | Main-axis alignment |
| `padding` | `'compact' \| 'normal' \| 'relaxed' \| 'loose'` | - | Internal padding |

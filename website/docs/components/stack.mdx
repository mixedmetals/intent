# Stack

The foundation of Intent layouts. Stack provides vertical and horizontal spacing with consistent, theme-aware gaps.

export const Stack = ({ children, direction = 'column', gap = 'normal', align, justify, padding, style, ...props }) => {
  const className = [
    'intent-stack',
    `intent-stack--direction-${direction}`,
    `intent-stack--gap-${gap}`,
    align && `intent-stack--align-${align}`,
    justify && `intent-stack--justify-${justify}`,
  ].filter(Boolean).join(' ');
  
  const paddingStyles = padding ? {
    compact: '4px',
    normal: '8px',
    relaxed: '16px',
    loose: '24px'
  }[padding] : undefined;
  
  return (
    <div className={className} style={{ padding: paddingStyles, ...style }} {...props}>
      {children}
    </div>
  )
}

export const Card = ({ children, padding = 'normal', style }) => (
  <div className={`intent-card intent-card--padding-${padding} intent-card--elevation-low`} style={style}>
    {children}
  </div>
)

export const Button = ({ children, importance = 'secondary', size = 'medium', ...props }) => (
  <button className={`intent-button intent-button--importance-${importance} intent-button--size-${size}`} {...props}>
    {children}
  </button>
)

## Overview

Stack is the most fundamental layout component in Intent. It handles spacing between child elements consistently across your application.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `direction` | `'column'` \| `'row'` | `'column'` | Stack direction |
| `gap` | `'compact'` \| `'normal'` \| `'relaxed'` \| `'loose'` | `'normal'` | Space between items |
| `padding` | `'compact'` \| `'normal'` \| `'relaxed'` \| `'loose'` \| `'none'` | `'none'` | Inner padding |
| `align` | `'start'` \| `'center'` \| `'end'` \| `'stretch'` | `'stretch'` | Cross-axis alignment |
| `justify` | `'start'` \| `'center'` \| `'end'` \| `'between'` \| `'around'` | `'start'` | Main-axis distribution |
| `wrap` | `boolean` | `false` | Allow items to wrap |

## Gap Values

| Value | Pixels | Usage |
|-------|--------|-------|
| `compact` | 4px | Tight groupings, icon buttons |
| `normal` | 8px | Standard spacing |
| `relaxed` | 16px | Form sections |
| `loose` | 24px | Page sections |

# Stat

Display a single statistic with label and trend.

## Example

<Grid columns={3} gap="normal" style={{ marginBottom: 24 }}>
  <Stat 
    label="Total Revenue"
    value="$45,231"
    trend={{ value: 12.5, direction: 'up' }}
  />
  <Stat 
    label="Active Users"
    value="2,345"
    trend={{ value: 8.2, direction: 'up' }}
  />
  <Stat 
    label="Bounce Rate"
    value="42.3%"
    trend={{ value: 2.1, direction: 'down' }}
  />
</Grid>

```tsx
import { Grid, Stat } from 'intent-react'

<Grid columns={3} gap="normal">
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | required | Statistic label |
| `value` | `string` \| `number` | required | Statistic value |
| `helperText` | `string` | - | Helper/description text |
| `icon` | `string` | - | Icon name |
| `size` | `'small'` \| `'medium'` \| `'large'` | `'medium'` | Stat size |
| `trend` | `{ value: number, direction: 'up' \| 'down' }` | - | Trend indicator |

# Stat

Display statistics and metrics. Uses the **Elevated** archetype.

## Examples

### Basic Stat

<div className="intent-demo">
  <div style={{ padding: '16px', border: '1px solid var(--intent-color-border)', borderRadius: '8px', background: 'var(--intent-color-surface)' }}>
    <div style={{ fontSize: '0.875rem', color: 'var(--intent-color-muted)', marginBottom: '4px' }}>Total Users</div>
    <div style={{ fontSize: '1.5rem', fontWeight: 600, marginBottom: '4px' }}>2,543</div>
    <div style={{ fontSize: '0.75rem', color: 'var(--intent-color-muted)' }}>All time registered users</div>
    <div style={{ fontSize: '0.875rem', color: 'var(--intent-color-success)', marginTop: '4px', fontWeight: 500 }}>↑ 12.5%</div>
  </div>
</div>

```tsx
import { Stat, StatLabel, StatValue, StatHelpText } from 'intent-react/components/data';

<Stat>
  <StatLabel>Total Users</StatLabel>
  <StatValue>2,543</StatValue>
  <StatHelpText>All time registered users</StatHelpText>
</Stat>
```

## Props

### Stat

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Stat content |

### StatLabel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Label text |

### StatValue

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Value text |

### StatHelpText

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Help text |

### StatGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Stat components |
| `columns` | `number` | - | Number of columns |

## Design Notes

- Uses **Elevated** archetype (subtle border)
- Label uses muted color
- Value uses large, bold text
- Trend indicators use success/error colors
- StatGroup uses auto-fit grid

# Switch

Toggle switch for binary on/off states.

## Examples

### Basic Switch

<Stack direction="row" gap="relaxed" style={{ marginBottom: 24 }}>
  <Switch label="Airplane mode" />
  <Switch label="Notifications" defaultChecked />
</Stack>

```tsx
<Switch label="Airplane mode" />
<Switch label="Notifications" defaultChecked />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | - | Switch label |
| `description` | `string` | - | Helper text |
| `size` | `'small'` \| `'medium'` \| `'large'` | `'medium'` | Switch size |
| `checked` | `boolean` | - | Controlled state |
| `defaultChecked` | `boolean` | `false` | Initial state |
| `disabled` | `boolean` | `false` | Disable switch |
| `loading` | `boolean` | `false` | Loading state |
| `onChange` | `(checked: boolean) => void` | - | Change handler |

## When to Use

Use **Switch** for:
- Instant toggle actions (no submit required)
- Settings that take effect immediately
- Binary on/off preferences

Use **Checkbox** for:
- Form submissions
- Multiple selections
- Terms acceptance

import React from 'react';
import { Switch } from 'intent-react';

# Switch

Toggle switch for boolean values. Uses the **milled archetype** with spring-physics thumb animation and a 0.5px sub-pixel hairline light-catcher.

export const SwitchDemo = ({ label, defaultChecked = false, disabled = false, size = 'md' }) => {
  const [checked, setChecked] = React.useState(defaultChecked)
  
  return (
    <Switch
      label={label}
      checked={checked}
      onChange={(e) => setChecked(e.target.checked)}
      disabled={disabled}
      size={size}
    />
  )
}

## Design Details

| Feature | Implementation |
|---------|---------------|
| **Archetype** | Milled — inset shadows, directional light |
| **Light Source** | 145° gradient on hairline and thumb |
| **Hairline** | 0.5px pseudo-element with `mask-composite: exclude` |
| **Thumb** | Metallic gradient, layered 3-depth shadow |
| **Animation** | Spring easing via `cubic-bezier(0.34, 1.56, 0.64, 1)` |
| **Active State** | Thumb width expands (stretch effect) |

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | - | Controlled value |
| `defaultChecked` | `boolean` | `false` | Default checked state |
| `label` | `ReactNode` | - | Switch label |
| `labelPosition` | `'left' \| 'right'` | `'right'` | Label placement |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size scale |
| `disabled` | `boolean` | `false` | Disable interaction |
| `onChange` | `ChangeEventHandler` | - | Native input change handler |

## CSS Output

```css
.intent-switch__track {
  position: relative;
  background: oklch(22% 0.012 275); /* OKLCH 0.12 chroma */
  box-shadow:
    inset 0 1px 2px 0 oklch(0% 0 0 / 0.30),
    inset 0 0 0 0.5px oklch(100% 0 0 / 0.06);
}

/* 0.5px hairline via mask-composite */
.intent-switch__track::before {
  content: '';
  position: absolute;
  inset: 0;
  border: 0.5px solid transparent;
  background: linear-gradient(145deg,
    oklch(100% 0 0 / 0.18),
    oklch(100% 0 0 / 0.04)) border-box;
  mask:
    linear-gradient(#fff 0 0) padding-box,
    linear-gradient(#fff 0 0) border-box;
  mask-composite: exclude;
}

.intent-switch__thumb {
  /* 145° metallic gradient */
  background: linear-gradient(145deg,
    oklch(100% 0 0) 0%,
    oklch(92% 0.005 275) 100%);
  box-shadow:
    0 0 0 0.5px oklch(0% 0 0 / 0.08),
    0 1px 2px 0 oklch(0% 0 0 / 0.15),
    0 3px 6px -1px oklch(0% 0 0 / 0.10);
  transition: transform 150ms cubic-bezier(0.34, 1.56, 0.64, 1);
}
```

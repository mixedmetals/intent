# Table

Display tabular data with sorting and selection. Uses the **Flat** archetype with subtle borders.

## Examples

```tsx
import { 
  Table, TableHeader, TableBody, TableRow, TableCell 
} from 'intent-react/components/data';

<Table>
  <TableHeader>
    <TableRow>
      <TableCell as="th">Name</TableCell>
      <TableCell as="th">Email</TableCell>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell>John Doe</TableCell>
      <TableCell>john@example.com</TableCell>
    </TableRow>
  </TableBody>
</Table>
```

### Striped Rows

```tsx
<Table variant="striped">
  <TableHeader>...</TableHeader>
  <TableBody>...</TableBody>
</Table>
```

### With Selection

```tsx
<Table 
  selectable
  selectedRows={selected}
  onSelectionChange={setSelected}
>
  ...
</Table>
```

### Sortable

```tsx
<Table>
  <TableHeader>
    <TableRow>
      <TableCell as="th" sortable sortDirection="asc" onSort={handleSort}>
        Name
      </TableCell>
    </TableRow>
  </TableHeader>
</Table>
```

## Props

### Table

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'striped'` | `'default'` | Table style |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Table size |
| `selectable` | `boolean` | `false` | Enable row selection |
| `selectedRows` | `string[]` | - | Selected row IDs |

### TableRow

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `selected` | `boolean` | `false` | Selected state |
| `disabled` | `boolean` | `false` | Disabled state |

### TableCell

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `as` | `'td' \| 'th'` | `'td'` | Cell element type |
| `align` | `'left' \| 'center' \| 'right'` | `'left'` | Cell alignment |
| `colSpan` | `number` | - | Column span |

## Design Notes

- Header uses **Elevated** background for distinction
- Cells use **Flat** archetype
- Borders use 1px `var(--intent-color-border)`
- Hover state on rows for better interaction
- Striped variant alternates row backgrounds

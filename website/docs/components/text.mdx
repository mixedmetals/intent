# Text

Typography component for consistent text styling.

## Examples

### Sizes

  <VStack gap="normal">

    <Text size="lg">Large - Lead paragraphs</Text>
    <Text size="xl">Extra Large - Subheadings</Text>
    <Text size="2xl">2XL - Section titles</Text>
  </VStack>
</div>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'xs' \| 'sm' \| 'base' \| 'lg' \| 'xl' \| '2xl' \| '3xl' \| '4xl'` | `'base'` | Text size |
| `weight` | `'normal' \| 'medium' \| 'semibold' \| 'bold'` | - | Font weight |
| `color` | `'default' \| 'muted' \| 'primary' \| 'success' \| 'warning' \| 'error'` | `'default'` | Text color |
| `align` | `'left' \| 'center' \| 'right'` | - | Text alignment |
| `truncate` | `boolean` | `false` | Truncate with ellipsis |
| `as` | `keyof JSX.IntrinsicElements` | `'span'` | HTML element to render |

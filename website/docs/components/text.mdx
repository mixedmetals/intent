# Text

Typography component for consistent text styling.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'xs'` \| `'sm'` \| `'base'` \| `'lg'` \| `'xl'` \| `'2xl'` \| `'3xl'` \| `'4xl'` | `'base'` | Font size |
| `weight` | `'normal'` \| `'medium'` \| `'semibold'` \| `'bold'` | `'normal'` | Font weight |
| `color` | `'default'` \| `'muted'` \| `'primary'` \| `'success'` \| `'warning'` \| `'danger'` | `'default'` | Text color |
| `align` | `'left'` \| `'center'` \| `'right'` | `'left'` | Text alignment |
| `truncate` | `boolean` | `false` | Single line with ellipsis |
| `as` | `'p'` \| `'span'` \| `'div'` \| `'label'` | `'span'` | HTML element |

## CSS Output

```css
.text {
  font-family: var(--intent-font-sans);
  line-height: 1.5;
}

.text--size-base {
  font-size: var(--intent-font-size-base);
}

.text--weight-medium {
  font-weight: 500;
}

.text--color-muted {
  color: var(--intent-color-text-muted);
}

.text--truncate {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}
```

# Timeline

Display chronological events. Uses the **Flat** archetype.

## Examples

### Basic Timeline

<div className="intent-demo">
  <Timeline>
    <TimelineItem 
      title="Pull Request Merged" 
      description="feat: add new table component"
      time="Today, 2:00 PM"
    />
    <TimelineItem 
      title="Commit Pushed" 
      description="fix: resolve sorting issue"
      time="Today, 10:30 AM"
    />
    <TimelineItem 
      title="Issue Created" 
      description="Bug: pagination not working"
      time="Yesterday, 4:15 PM"
    />
  </Timeline>
</div>

## Props

### Timeline

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `alternate` | `boolean` | `false` | Alternate item alignment |

### TimelineItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | - | Item title |
| `description` | `string` | - | Item description |
| `time` | `string` | - | Timestamp |
| `icon` | `ReactNode` | - | Custom icon |

## Design Notes

- Uses **Flat** archetype (no bevel)
- Vertical line connects items
- Dot indicator at each item
- Last item has no connecting line
- Icons can be emoji or custom components

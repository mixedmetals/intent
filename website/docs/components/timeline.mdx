# Timeline

Chronological display of events or activities.

## Example

<Timeline 
  items={[
    {
      id: 1,
      title: 'Project Created',
      description: 'The project was initialized by John Doe',
      timestamp: '2024-01-15 10:30 AM',
      icon: 'plus-circle',
      color: 'success'
    },
    {
      id: 2,
      title: 'First Commit',
      description: 'Initial codebase pushed to repository',
      timestamp: '2024-01-15 2:45 PM',
      icon: 'git-commit',
      color: 'primary'
    },
    {
      id: 3,
      title: 'Team Member Added',
      description: 'Jane Smith was added as a collaborator',
      timestamp: '2024-01-16 9:00 AM',
      icon: 'user-plus',
      color: 'info'
    },
    {
      id: 4,
      title: 'Deployment Failed',
      description: 'Build failed due to missing environment variables',
      timestamp: '2024-01-16 4:20 PM',
      icon: 'alert-circle',
      color: 'danger'
    },
  ]}
  style={{ marginBottom: 24 }}
/>

## With Custom Content

```tsx
<Timeline 
  items={[
    {
      id: 1,
      title: 'New Comment',
      content: (
        <Card padding="normal">
          <Text>"Great work on this feature!"</Text>
          <Text size="sm" color="muted">- Jane Doe</Text>
        </Card>
      ),
      timestamp: '5 minutes ago'
    },
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `TimelineItem[]` | required | Timeline events |
| `variant` | `'default'` \| `'compact'` \| `'horizontal'` | `'default'` | Timeline style |

### TimelineItem Type

```typescript
interface TimelineItem {
  id: string | number
  title: string
  description?: string
  content?: ReactNode
  timestamp?: string
  icon?: string
  color?: 'default' | 'primary' | 'success' | 'warning' | 'danger' | 'info'
}
```

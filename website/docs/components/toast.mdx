# Toast

Temporary notification that appears and dismisses automatically.

## Example

```tsx
import { useState } from 'react'
import { Toast, Button } from 'intent-react'

function Example() {
  const [show, setShow] = useState(false)
  
  return (
    <>
      <Button onClick={() => setShow(true)}>
        Show Toast
      </Button>
      
      {show && (
        <Toast
          variant="success"
          title="Success!"
          onDismiss={() => setShow(false)}
        >
          Your changes have been saved.
        </Toast>
      )}
    </>
  )
}
```

## Variants

```tsx
<Toast variant="info" title="Info">
  Here's some helpful information.
</Toast>

<Toast variant="success" title="Success!">
  Your action completed successfully.
</Toast>

<Toast variant="warning" title="Warning">
  Please review before continuing.
</Toast>

<Toast variant="danger" title="Error">
  Something went wrong. Try again.
</Toast>
```

## With Action

```tsx
<Toast 
  variant="info"
  title="New update available"
  action={{ label: 'Update', onClick: handleUpdate }}
>
  Version 2.0 is ready to install.
</Toast>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'info'` \| `'success'` \| `'warning'` \| `'danger'` | `'info'` | Toast style |
| `title` | `string` | - | Toast heading |
| `action` | `{ label: string, onClick: () => void }` | - | Action button |
| `duration` | `number` | `5000` | Auto-dismiss time (ms) |
| `onDismiss` | `() => void` | - | Dismiss callback |

## Toast Provider Pattern

For multiple toasts, use a provider:

```tsx
// App.tsx
import { ToastProvider } from 'intent-react'

function App() {
  return (
    <ToastProvider>
      <YourApp />
    </ToastProvider>
  )
}

// Any component
import { useToast } from 'intent-react'

function MyComponent() {
  const toast = useToast()
  
  const handleSave = () => {
    saveData()
    toast.success({
      title: 'Saved!',
      description: 'Your changes were saved.'
    })
  }
  
  return <Button onClick={handleSave}>Save</Button>
}
```

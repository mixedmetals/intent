# Tooltip

Hover-activated information popup. Uses the **Elevated** archetype for the tooltip panel.

## Examples

### Basic Tooltip

Hover over the button:

<div className="intent-demo">
  <Tooltip content="Click to save your changes">
    <Button importance="primary">Hover Me</Button>
  </Tooltip>
</div>

```tsx
import { Tooltip, Button } from 'intent-react';

<Tooltip content="Click to save your changes">
  <Button>Hover Me</Button>
</Tooltip>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `content` | `ReactNode` | - | Tooltip content |
| `position` | `'top' \| 'right' \| 'bottom' \| 'left'` | `'top'` | Tooltip position |
| `delay` | `number` | `200` | Show delay in ms |
| `children` | `ReactElement` | - | Target element (must accept refs) |

## Design Notes

- Tooltip uses **Elevated** archetype (floating above content)
- Dark background with light text for contrast
- Small border radius (6px)
- Arrow points to trigger element
- Respects safe area boundaries

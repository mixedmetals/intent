import { Stack, Button, Tooltip } from 'intent-react'

# Tooltip

Hover-activated information popup.

## Examples

### Basic Tooltip

Hover over the button:

<Stack direction="row" gap="normal" style={{ marginBottom: 24 }}>
  <Tooltip content="Click to save your changes">
    <Button>Hover Me</Button>
  </Tooltip>
</Stack>

```tsx
<Tooltip content="Click to save your changes">
  <Button>Hover Me</Button>
</Tooltip>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `content` | `string` \| `ReactNode` | - | Tooltip content |
| `position` | `'top'` \| `'right'` \| `'bottom'` \| `'left'` | `'top'` | Tooltip position |
| `delay` | `number` | `200` | Show delay in ms |
| `disabled` | `boolean` | `false` | Disable tooltip |

## Accessibility

- Uses `aria-describedby` to associate with trigger
- Trigger element must be focusable
- Keyboard accessible (shows on focus)

## When to Use

**Use Tooltips for:**
- Icon button labels
- Abbreviation explanations
- Additional context that isn't critical

**Don't use for:**
- Critical information (use Alert)
- Content that needs to be copied (use Popover)
- Mobile (tooltips don't work well on touch)

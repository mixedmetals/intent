# Tree

Hierarchical tree view component. Uses the **Flat** archetype for items.

## Examples

      </TreeBranch>

### Default Expanded

```tsx
<Tree defaultExpanded={['src', 'components']}>
  <TreeItem id="src" label="src">
    <TreeBranch>
      <TreeItem id="components" label="components">
        <TreeBranch>
          <TreeItem label="Button.tsx" />
        </TreeBranch>
      </TreeItem>
    </TreeBranch>
  </TreeItem>
</Tree>
```

### Selectable

```tsx
<Tree 
  selectable 
  selectedIds={['item-1']}
  onSelectionChange={setSelected}
>
  <TreeItem id="item-1" label="Item 1" />
  <TreeItem id="item-2" label="Item 2" />
</Tree>
```

## Props

### Tree

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultExpanded` | `string[]` | `[]` | Default expanded node IDs |
| `expandedIds` | `string[]` | - | Controlled expanded IDs |
| `selectable` | `boolean` | `false` | Enable selection |
| `selectedIds` | `string[]` | - | Selected node IDs |
| `onSelectionChange` | `(ids: string[]) => void` | - | Selection change callback |

### TreeItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | - | Item label |
| `icon` | `ReactNode` | - | Item icon |
| `id` | `string` | - | Unique identifier |

## Design Notes

- Uses **Flat** archetype (no bevel)
- Indentation shows hierarchy level
- Chevron rotates when expanded
- Hover state on items
- Selected state uses primary color
